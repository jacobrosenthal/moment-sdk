Line

Moment. Line

new Line(p1, p2, effect)

Source:

Represents a transition between two points of vibration - a straight line is drawn between the two points using a pre-defined effect for transition.

Parameters:
Name Type Description
p1 Point

the first point
p2 Point

the second point
effect Effect

the effect to use for transitioning
