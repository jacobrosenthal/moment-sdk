Timeline

Moment. Timeline

new Timeline(vibrations)

Source:
  • timeline.js, line 33

Represents a sequence of vibrations arranged in a timeline that can be evaluated by Moment's internal haptic drivers.

Parameters:
Name Type Description
vibrations Array

The sequence of vibrations to place in timeline

Methods

addDelay(delay)

Source:
  • timeline.js, line 164

Adds a delay to all vibrations in the timeline. A negative number will reduce the delay.

Example
var t = new Moment.Timeline([v1, v2, v3, v4, v5]);
t.addDelay(500); // adds 500ms delay to all vibrations
t.start(); // starts the vibrations v1...v5
Parameters:
Name Type Description
delay Number

The number of milliseconds to add to the delay

clone() → {Timeline}

Source:
  • timeline.js, line 120

Returns a clone of the timeline object with the same vibration sequence, allowing direct manipulation of the timeline without modifying the original object.

Example
var t = new Moment.Timeline([v1, v2, v3]);
var n = t.clone().push(v4); // contains v1...v4
t.includes(v4); // false
Returns:

a new Timeline clone of the original object

Type
Timeline

Appends a vibration to the timeline object.

Example
var t = new Moment.Timeline([v1, v2]); // timeline with v1 and v2
t.push(v3); // t now contains v1, v2, and v3

splice() → {Timeline}

Source:
  • timeline.js, line 52

Similar to Array.prototype.splice(), this allows insertion and deletion of items at a specific index in the timeline.

Example
var t = new Moment.Timeline([v1, v2, v3]);
t.splice(1, 1, v4, v5); // t now contains [v1, v4, v5, v3]
Returns:
  • this, a chainable return value
Type
Timeline

totalTime() → {Number}

Source:
  • timeline.js, line 198

Compute the total time required to execute the timeline of vibrations.

Example
var t = new Moment.Timeline([v1, v2, v3]);
t.totalTime(); // 1500ms
t.addDelay(500); // adds 500ms delay to all vibrations
t.totalTime(); // 2000ms
Returns:

the total time required to execute the timeline

Type
Number
